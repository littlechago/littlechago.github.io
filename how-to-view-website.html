How to View QR Hunt Website

This guide will help you properly view and test the QR Hunt website on your local computer.

Option 1: Using the Test Navigation Page

The simplest way to test the website is to use the test navigation page:

  1. Open the test-navigation.html file in your browser
  2. Use the links on that page to navigate to different parts of the website
  3. Test that all the links and functionality work correctly
Open Test Navigation

Option 2: Opening Files Directly

You can open the HTML files directly in your browser:

  • index.html - The main QR game page (landing page)
  • about.html - Information about QR Hunt
  • ad-landing-simple.html - The ad landing page

Note: When opening files directly, some features might not work correctly due to browser security restrictions. For the best experience, use a local web server (Option 3).

Option 3: Using a Local Web Server (Recommended)

For the best experience, you should use a local web server to view the website. Here are some options:

Using Python (if installed):

  1. Open a command prompt or terminal
  2. Navigate to the folder containing the website files
  3. Run one of these commands:
    • Python 3: python -m http.server 8000
    • Python 2: python -m SimpleHTTPServer 8000
  4. Open your browser and go to http://localhost:8000

Using Node.js (if installed):

  1. Open a command prompt or terminal
  2. Navigate to the folder containing the website files
  3. Run: npx serve
  4. Open the URL shown in the terminal

Using a simple web server extension:

You can install a web server extension for your browser:

  • For Chrome: Web Server for Chrome
  • For VS Code: Use the "Live Server" extension

Troubleshooting

  • QR codes not loading: This might be due to browser security restrictions when opening files directly. Try using a local web server.
  • JavaScript errors: Check the browser console (F12) for any error messages.
  • Links not working: Make sure all file paths are correct and the files exist in the correct locations.

Remember: The QR game is now the landing page (index.html), and the play.html page redirects to index.html.
